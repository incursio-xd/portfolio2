# Portfolio 2.0

## Description
Portfolio 2.0 is a modern personal website built with React, Vite, Tailwind CSS, and other cutting-edge technologies. It showcases my skills, projects, and professional experience with a sleek, interactive design. The portfolio features smooth animations, a responsive layout, and an integrated contact form for seamless communication.

## Demo
Check out the live version of my portfolio **[here](https://incursio-xd.github.io/portfolio2/)**.

## Technologies Used
* **React** - Component-based UI library
* **Vite** - Next-generation frontend tooling
* **Tailwind CSS** - Utility-first CSS framework
* **Framer Motion** - Animation library for React
* **React Hook Form** - Form validation and management
* **EmailJS** - Email service integration
* **PostCSS** - CSS transformations

## Features
* **Showcase of Skills**: Display of technologies and technical expertise
* **About Section**: Information about my background and experience
* **Projects Gallery**: Highlighting key projects with descriptions
* **Experience Timeline**: Professional work history
* **Contact Form**: Easy communication through integrated email service
* **Responsive Design**: Optimized for all screen sizes
* **Smooth Animations**: Engaging transitions and effects using Framer Motion
* **Fast Performance**: Lightning-fast load times with Vite
* **Modern UI**: Clean, contemporary design with Tailwind CSS

## Installation
To run Portfolio 2.0 locally, follow these steps:

1. Clone the repository:
```bash
git clone https://github.com/incursio-xd/portfolio2.git
```

2. Navigate to the project directory:
```bash
cd portfolio2
```

3. Install the dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm run dev
```

5. Open your browser and visit **http://localhost:5173** to see the portfolio in action.

## Build for Production
To create an optimized production build:
```bash
npm run build
```

To preview the production build locally:
```bash
npm run preview
```
